About This Flask GraphQL App

This is a modern Flask application with GraphQL API and MongoDB backend featuring:

Backend Features

  • Flask with GraphQL API
  • MongoDB with MongoEngine ODM
  • Graphene-Flask for GraphQL integration
  • Blueprint-based organization
  • CORS support
  • GraphiQL interactive playground

Frontend Features

  • Bootstrap 5 for styling
  • Responsive design
  • Template inheritance
  • GraphQL client functionality
  • Modern UI components
  • Interactive forms

GraphQL Schema

The API supports the following operations:

Queries
  • users - Get all users
  • user(id) - Get user by ID
  • health - Health check
Mutations
  • createUser - Create new user
  • updateUser - Update existing user
  • deleteUser - Delete user
Example GraphQL Query
{
  users {
    id
    username
    email
    createdAt
  }
}
Example GraphQL Mutation
mutation {
  createUser(username: "john_doe", email: "john@example.com") {
    user {
      id
      username
      email
    }
    success
    error
  }
}
